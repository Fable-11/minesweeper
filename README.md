# Minesweeper

A Minesweeper game built with Pygame featuring a hand-crafted pixel art aesthetic.

## Features

- 4 switchable themes: Classic, Autumn Harvest, Midnight Moon, Lavender Haze
- Unique pixel art flag per theme (red "1", falling leaf, eclipse, butterfly)
- Flag placement animations per theme
- Themed flag removal animations: leaf-fall crumble (Autumn), sweep bar wipe (Midnight), flutter dissolve (Lavender)
- Theme-specific sound sets (clicks, flags, explosions, win jingles, difficulty buttons)
- Pixel art aesthetic with custom color palettes per theme
- 3 difficulty modes: Easy (9x9), Medium (16x16), Expert (30x16)
- Programmatic sound effects — all generated in code, no audio files
- Classic number colors (blue, green, red, navy, maroon, teal)
- Best time leaderboard saved to JSON
- Win/lose animations
- Theme preference saved between sessions
- Single file, no external assets required

## How to Play

```
pip install pygame
python minesweeper.py
```

- **Left click** — Reveal a cell
- **Right click** — Place/remove a flag
- First click is always safe
